## **Project OverView**
---
 - **Description:** Αυτο το Spring Boot API λειτουργεί ως το κεντρικό backend συστημά </br> που αντλεί και επεξεργάζεται λειτουργικά δεδομένα απο τον SQL Server.
 - **Tech Stack:** Java 21, Spring Boot 2, SQL Server, Maven Manager
 - **Main Goal:** Η αυτοματοποιημένη τροφοδότηση των αποτελεσμάτων ψηφοφορίας σε </br>  
Power BI ωστε η ομαδα ανάλυσης να τροφοδοτεί το προεδρείο.
