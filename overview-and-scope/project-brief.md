# 1. Overview & Scope 

## 1.1 Executive Summary 
* **What it is:** It is a dashboard tailored to the LC's needs.
* **The Core Value:** It centralizes and maintains our personal information in a database designed by us , for us and securely living in the cloud. It provides Event Announcements, Polls and Voting abilities to authorized users, Reports for the Board Members.

## 1.2 Problem & Objectives 
* **The Current State:** Personal Information scattered across platforms, decentralized Attendance Check and Voting Procedure.
* **the Desired State:** Sheltered Personal Information, organized in a Relational Database. Automated Voting Procudure directly linked to Attendance Check. A dashboard for the eestecer.

## 1.3 Target Audience & User Personas
* **Primary Users:** The primary Users of this platform are the members of EESTEC LC Thessaloniki, with emphasis on the Board.
* **Secondary Users:** Junior Members that dont have the ability to vote.

## 1.4 Project Scope (Boundaries)
### **In Scope:**
* **User Authentication via JWT keys**
* **CRUD operations with SpringBoot**
* **ReactJS in the WebApp**
